# Yemedhnat Internal Medicine Clinic Website

A stunning, modern Next.js website for Yemedhnat Internal Medicine Specialty Clinic. Built with a focus on aesthetics, mobile responsiveness, and user experience.

## Features

- **Modern Design**: Calming, trustworthy aesthetic with the clinic's blue color palette
- **Smooth Animations**: Framer Motion powered animations throughout
- **Mobile-First**: Fully responsive design optimized for mobile devices
- **Performance**: Optimized for Lighthouse scores 90+
- **Accessibility**: WCAG 2.1 AA compliant with semantic HTML

## Tech Stack

- **Next.js 14+** with App Router
- **Tailwind CSS** for styling with custom theme
- **Framer Motion** for smooth animations
- **Lucide React** for icons
- **React Hook Form** for form handling
- **TypeScript** for type safety

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
